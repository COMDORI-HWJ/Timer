## ✨ 특징
1. 타이머를 밀리초 단위(00:00:00:000)로 카운트 다운을 할 수 있다.
2. 타이머를 밀리초 단위로 시간 조절이 가능하다.
3. 타이머 시간을 숫자로 입력할 수 있다.
4. 타이머가 끝나면 푸시 알림으로 알려주며 푸시 알림을 탭 하였을 때 타이머 화면으로 이동한다.
5. 백그라운드 타이머를 사용할 수 있다.(다시 앱을 켜도 이어서 작동)
6. 밀리초 단위 스톱워치를 사용할 수 있다.
7. 버그 제보 및 아이디어 추가 문의 피드백을 보낼 수 있다.
8. 처음 앱을 설치하면 튜토리얼(앱 사용 방법)이 자동으로 표시되어 사용자가 도움받을 수 있다.
9. 시스템 설정에 맞춰서 화면 스타일(라이트 모드 / 다크 모드)이 자동으로 적용된다.
10. 앱 평가를 설정에서 남길수 있다.
